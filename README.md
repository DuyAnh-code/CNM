## 🚀 Bắt đầu

1. Cài đặt dependencies:

```bash
npm install
```

2. Sao chép `.env.example` thành `.env.local` và điền thông tin Supabase:

```bash
cp .env.example .env.local
```

3. Chạy development server:

```bash
npm run dev
```

4. Mở [http://localhost:3000](http://localhost:3000) trên trình duyệt.
